## 上傳方式

1. 到 GitHub 建立新的 Public repository：sport-profit-app
2. 上傳本資料夾所有檔案到 repository 根目錄
3. 到 Settings → Pages
4. Source 選 Deploy from a branch
5. Branch 選 main，Folder 選 /(root)
6. Save

GitHub Pages 發布完成後，用 iPhone Safari 打開網址，再選「分享 → 加入主畫面」。
