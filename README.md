# 運彩衝業績助手 正式版 V1.0

GitHub Pages 網址：
https://chidachen.github.io/sport-profit-app/

## 正式版功能

### 第一頁：業績計算
- 目標業績
- 客隊賠率
- 主隊賠率
- 固定利潤率 6.25%
- 獎金支出率
- 客隊下注
- 主隊下注
- 客勝回收
- 主勝回收
- 對沖損失
- 利潤
- 損失
- 損失率

### 第二頁：賠率比較
- 有格式輸入，不用貼文字
- 預設三組賠率：
  - 第1組：1.75 / 1.75
  - 第2組：1.74 / 1.76
  - 第3組：1.72 / 1.72
- 可新增賠率組
- 一鍵載入常用賠率
- 自動依損失由低到高排序

## iPhone 安裝方式

1. 用 Safari 開啟：
   https://chidachen.github.io/sport-profit-app/
2. 點分享按鈕
3. 選「加入主畫面」
4. 名稱輸入「運彩衝業績助手」
5. 按加入

## 更新方式

解壓縮本 ZIP 後，把裡面的所有檔案上傳到 GitHub repository 根目錄，按 Commit changes。
請不要上傳整個資料夾。
